## 사용자 입력 요약
- 장르 감성: 다크 판타지
- 진행 구조: 층을 내려가는 던전 탐사
- 플레이 루프:
  - 캐릭터는 칸 단위로 이동한다.
  - 플레이어가 행동할 때마다 몬스터도 행동한다.
  - 플레이어가 한 칸 이동하면 몬스터는 플레이어를 향해 한 칸 이동하거나, 공격 가능 시 공격한다.
  - 던전 내부에서 무기와 아이템을 습득하고 즉시 사용 가능하다.
  - 체력이 모두 소진되면 1층으로 돌아온다.
  - 사망 시 보유 아이템은 모두 소실된다.
- 확장 아이디어:
  - 반복 플레이 중에도 보존 가능한 재화를 모아 업그레이드하는 구조를 고려 중
  - 던전 하층 진행 과정에서 새로운 플레이어블 캐릭터를 해금하는 구조를 고려 중
- 추가 구체화 요청:
  - 재화는 2종으로 확정
  - 던전 내부 재화 1종은 런 중에만 사용하며, 던전 내부 상인과 이벤트에서 아이템으로 교환한다.
  - 해당 재화로 얻은 아이템은 이번 던전런이 끝나면 소멸한다.
  - 던전 외부 재화 1종은 거점에서 업그레이드, 해금, 출발 준비에 사용한다.
  - 던전 외부 재화는 던전 진입 전 아이템 풀 확장, 시작 보급품 구매, 출발 장비 준비에 사용한다.
- 추가 확정 요청:
  - 플레이어 개인 동기 한 문장 확정
  - `잔광`의 첫 빌드 사용처 1종 확정
  - `혈편`의 첫 빌드 사용처 1종 확정
  - 거점 1차 기능 목록 확정
  - `MQ-01` 초반 목표 문구 확정

## 설계 목적
- 반복 탐사와 사망 회귀 구조가 서사적으로 자연스럽게 설명되는 세계관 구축
- 아이템 소실, 1층 회귀, 영구 재화, 캐릭터 해금이 모두 같은 설정 안에서 연결되도록 정리
- 이후 `02_quest_design.md`, `03_dialogue_script.md`, `04_branch_map.md`로 확장 가능한 기반 확보

## 명시적 가정
- 던전은 단순한 물리 공간이 아니라 저주, 봉인, 순환 의식 같은 초자연적 규칙에 의해 유지된다.
- 1층 회귀는 단순 게임 규칙이 아니라 세계관 내부에서 설명 가능한 현상이어야 한다.
- 영구 재화는 "기억", "영혼 파편", "봉인의 잔재"처럼 사망 후에도 남는 비물질적 자원으로 설정하는 것이 적합하다.
- 캐릭터 해금은 단순 모집이 아니라 던전에 묶인 존재를 해방하거나 계약을 맺는 구조가 적합하다.
- 전투 수치, 적 AI 상세 규칙, 맵 레이아웃은 이번 문서 범위에서 제외한다.

- 플레이 루프를 지지하는 서사적 장치
- 추후 퀘스트와 캐릭터 확장에 필요한 세계관 키워드

## 후속 작업 진행 기준
- 기존 `01_worldbuilding.md`의 핵심 규칙을 고정값으로 사용한다.
- 후속 문서는 다음 축을 유지해야 한다.
  - 회귀는 축복이 아니라 침식성 저주다.
  - 하강은 단순 탐사가 아니라 진실과 죄업을 파헤치는 행위다.
  - 세력 갈등의 중심은 권력 다툼보다 회귀와 성혈을 어떻게 해석하는가에 있다.
  - 캐릭터 해금은 구조가 아닌 서사적 해방과 계약으로 설명한다.
- 이번 후속 단계에서 확정할 항목:
  - 플레이어 개인 동기 문장
  - `잔광` 1차 사용처
  - `혈편` 1차 사용처
  - 거점 1차 기능 목록
  - `MQ-01` 초반 목표 문구
