# 리뷰 보고서

## 검토 범위

  - `01_worldbuilding.md`
  - `02_quest_design.md`
  - `03_dialogue_script.md`

- 검토 관점:
  - 설정 정합성
  - 퀘스트 동기와 반복 구조 적합성
  - 대사 톤 일관성
  - 분기 추적 가능성

## 총평
- 현재 문서 세트는 `회귀의 저주`, `하강을 통한 진실 접근`, `영구 재화의 서사적 정당화`, `캐릭터 해금의 해방 구조`를 비교적 일관되게 공유하고 있다.
- 메인 퀘스트, 대사, 분기 모두가 `낙죄의 성혈`을 단순 던전이 아니라 죄와 기억을 재구성하는 장치로 다루고 있어 세계관 중심축은 안정적이다.
- 세력 갈등도 단순 선악 구도가 아니라 회귀와 성혈의 해석 차이로 정리되어 있어 후속 확장 여지가 충분하다.

## 확인된 강점
- 아이템 소실, 회귀, 영구 재화가 하나의 설정 규칙으로 자연스럽게 묶여 있다.
- 캐릭터 해금이 구조적 보상에 머무르지 않고 각 인물의 속박 해소와 연결되어 있다.
- 엔딩 분기가 세력 우호도보다 세계 해석의 방향성에 기반해 있어 테마 일관성이 높다.
- 대사 톤이 세력과 인물 성격에 맞게 구분되어 있어 후속 장면 확장 시 기준점이 된다.

## 리스크 및 보완 필요 사항

### 1. 메인 퀘스트의 플레이 동기 선명도
- 문제:
  - `MQ-02` 이후 플레이어가 왜 계속 하강해야 하는지 정서적 동력이 다소 약해질 수 있다.
- 영향:
  - 세계 존속 문제는 크지만, 개인적 동기가 약하면 반복 탐사 구조에서 몰입이 흔들릴 수 있다.
- 제안:
  - `MQ-01` 또는 `MQ-04`에서 플레이어 자신의 상실 대상 하나를 더 선명하게 고정한다.
  - 예: 잃어버린 이름, 특정 인물, 이전 루프의 결말 기록.

### 2. 세력 퀘스트 보상 구분의 추가 필요
- 문제:
  - 현재 세력별 보상 방향은 분리되어 있으나, 실제 플레이 체감 차별화 기준은 더 필요하다.
- 영향:
  - 선택의 의미가 서사적으로는 살아도 체감상 비슷해질 수 있다.
- 제안:
  - 수도회는 침식 완화와 정보 공개
  - 흑철 잔존단은 생존성과 전투 교리
  - 연금회는 개조와 위험한 효율
  - 무명 계약자는 탐색/루프 변형
  - 위와 같이 플레이 성향 차이를 명시적으로 더 벌리는 것이 좋다.

### 3. 기억 관련 분기 설명의 단계적 공개 필요
- 문제:
  - `기억 보존`과 `기억 소거` 축이 세계관상 중요하지만, 초반부터 과도하게 제시되면 추상적으로 느껴질 수 있다.
- 영향:
  - 플레이어가 선택 의미를 이해하기 전에 철학적 선택만 남을 위험이 있다.
- 제안:
  - 초반은 개인 정체성 중심
  - 중반은 성혈 시스템 이해 중심
  - 후반은 윤리적 선택 중심
  - 이렇게 공개 단계를 나누는 편이 적합하다.

### 4. 엔딩 E의 활용도 관리 필요
- 문제:
  - `공허한 반복` 엔딩은 세계관적으로 타당하지만, 준비가 부족하면 단순한 미달 엔딩처럼 보일 수 있다.
- 영향:
  - 플레이어가 실패했다고만 느끼면 서사적 여운이 약해진다.
- 제안:
  - END-E에도 고유 컷신, 반복된 제단 대사 변화, 정체성 붕괴 연출을 넣어 독립된 결말처럼 보강한다.

## 문서 간 충돌 여부
- 명시적 충돌은 현재 기준으로 없음.
- 다음 항목은 후속 제작 시 주의가 필요하다.
  - `03_dialogue_script.md`에서 수도회 대사가 지나치게 진실을 직접 설명하면 `04_branch_map.md`의 후반 진실 공개 흐름이 약해질 수 있다.
  - `02_quest_design.md`의 캐릭터 해금 퀘스트가 너무 빠르게 완료되면 반복 탐사의 축적형 구조가 약해질 수 있다.
  - `04_branch_map.md`의 엔딩 판정은 세력 상태값보다 신념 플래그 우세를 중심으로 유지해야 현재 테마가 흔들리지 않는다.
