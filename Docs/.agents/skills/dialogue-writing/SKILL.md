# Dialogue Writing



게임 대사 작성을 위한 실행 스킬이다.



## 목적



이 스킬은 다음 작업에 사용한다.



- NPC 대사 작성

- 플레이어 선택지 작성

- 컷신 대사 작성

- 바크(Bark) 작성

- 감정 태그 및 연기 가이드 작성

- 상황별 대화 흐름 구조화

## 작업 원칙



- 세계관과 퀘스트를 먼저 참조한다.

- 캐릭터별 말투를 일관되게 유지한다.

- 대사는 정보 전달, 성격 표현, 감정 유발을 동시에 달성해야 한다.

- 4번째 벽을 깨는 표현을 피한다.

- 선택지는 결과를 암시하되 지나치게 노골적이지 않게 설계한다.

- 반복 대사는 최소 3개 이상의 변형을 고려한다.



## 작업 절차



1. 등장인물의 성격과 말투를 정리한다.

2. 컷신 대사와 주요 대화 장면을 작성한다.

3. 퀘스트 관련 NPC 대사를 작성한다.

4. 플레이어 선택지와 선택 후 반응 대사를 작성한다.

5. 전투, 탐색, 관계 변화에 따른 바크를 작성한다.

6. 분기 설계와 연결될 선택지를 식별한다.

## 품질 기준



- 캐릭터별 보이스가 일관되어야 한다.

- 상황과 감정선이 대사에 반영되어야 한다.

- 선택지는 플레이어 의도를 분명하게 구분해야 한다.

- 반복 대사는 기계적으로 느껴지지 않아야 한다.

- 분기와 연결되는 선택지는 후속 구조 설계가 가능해야 한다.



## 에러 핸들링



### 캐릭터 설정이 부족한 경우

- 퀘스트 맥락과 역할을 기반으로 성격을 추론한다.

- 보강이 필요한 캐릭터는 문서에 명시한다.



### 대사량이 과도한 경우

- 핵심 컷신과 핵심 NPC부터 우선 작성한다.

- 나머지는 추가 작성 필요 항목으로 표시한다.
